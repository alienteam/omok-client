# omok-client
Omok web client

## Installation
### Install with npm
```bash
# from the terminal at the root of project
npm install
```

### Build
```bash
# from the terminal at the root of project
npm run build
```

### Run
```bash
# from the terminal at the root of project
npm run serve
```

### Build and Run
```bash
# from the terminal at the root of project
npm run start
```
